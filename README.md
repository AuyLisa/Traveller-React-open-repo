# 🧳 Traveller — туры, отели и авиабилеты в одном месте

**Traveller** — это учебный проект интернет-магазина туристических услуг. Статическое SPA-приложение на React с клиентской маршрутизацией, корзиной, избранным и личным кабинетом.

🔗 **Деплой:** [https://traveller-react.vercel.app/]

## 🛠️ Технологии

| Технология | Назначение |
|------------|------------|
| **React 18** | Библиотека для построения пользовательских интерфейсов |
| **React Router v6** | Клиентская маршрутизация (SPA) |
| **Context API** | Глобальное состояние (пользователь, корзина, избранное) |
| **localStorage** | Персистентное хранение данных (авторизация, корзина, избранное) |
| **Vite** | Сборщик проекта (быстрая разработка и сборка) |
| **CSS (БЭМ)** | Стилизация компонентов (без препроцессоров) |
| **Flexbox / Grid** | Построение сеток и адаптивных макетов |
| **Алиасы (alias)** | Короткие пути для импортов (`@ui`, `@hooks`, `@utils` и др.) |

### React Hooks
- `useState` — локальное состояние компонентов
- `useEffect` — синхронизация с `localStorage`
- `useContext` + кастомный хук `useUser()` — доступ к данным пользователя
- `useMemo`, `useCallback` — оптимизация производительности

## 🗺️ Структура страниц

| Страница | Описание |
|----------|----------|
| **Главная (Рекомендации)** | Три раздела: популярные направления, майские праздники, туры в отели |
| **Туры** | Фильтрация по стране, городу, звёздам, ночам, цене; сортировка; карточки с галереей и кнопка добавить в избранное |
| **Авиабилеты** | Фильтрация по городам, авиакомпании, цене, длительности; выбор класса (эконом/комфорт/бизнес) |
| **Отели** | Фильтрация по стране, городу, звёздам, отзывам, цене; карточки с галереей |
| **Подробный тур / отель / номер** | Галерея, описание, характеристики, кнопка добавления в корзину |
| **Корзина** | Все добавленные товары с изменением количества и удалением |
| **Оформление заказа** | Форма с паспортными данными и контактами |
| **Личный кабинет** | Профиль, корзина, избранное, история бронирований |
| **Избранное** | Лайкнутые туры, отели, авиабилеты и номера (сортировка по типам) |
| **О нас** | Информация о компании, преимуществах и контактах |
| **Вход / Регистрация** | Авторизация с сохранением в `localStorage` |

## ⚡ Ключевые особенности

### 🔐 Управление пользователем (UserContext)
- Авторизация → данные сохраняются в `localStorage` и глобальном состоянии
- При перезагрузке страницы сессия восстанавливается
- Выход удаляет данные из состояния и хранилища

### 🛒 Корзина (CartContext)
- Товары добавляются независимо от авторизации
- Поддерживаются типы: `trip`, `flight`, `room`
- Количество меняется через контролы `+ / -`

### ❤️ Избранное (Favorites)
- Лайки сохраняются отдельно для каждого пользователя
- Ключи в `localStorage`: `favorites_user_{id}`
- Поддерживаются типы: `trip`, `hotel`, `avia`, `room`
- На странице избранного товары сортируются: туры → отели → номера

### 📱 Адаптивная вёрстка (mobile-first)
- Базовые стили для мобильных устройств
- Медиазапросы для планшетов (768px+) и десктопов (1024px+)
- `clamp()` для резиновых размеров шрифтов и отступов
- Flexbox и Grid для построения сеток
